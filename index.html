The way this works:
You get a function "CreateWindow()"
It has the parameters ({position}, {options}) which are both Javascript objects
The first window created should be to the right and is made using function:

CreateWindow({x:200,y:550}, {parentid:"canvas",resizable:true,movable:true, title:"Window", dragArea:"title"})

Position =
{
0 x: integer || default 15px
1 y: integer || default 15px
}

Options =
{
0 parentid: string || default body
1 movable: bool || default true
2 resizable: bool || default true
3 text: string || default ""
4 dragArea: option || full or title || default full
5 title: string || default ""
6 titleColor: hex || default #AEC6CF
}
